Structure-based Kinases Resources

The Structure-based Kinase Resources is constructed by collecting all the published kinase domain PDB structures, containing four components: Kinase Domain PDB database (KPDB), Kinase Ligand-binding PDB database (KLPDB), Apo Kinase PDB database (AKPDB), and Covalent Inhibitor-binding Kinase PDB database (CIKPDB).
