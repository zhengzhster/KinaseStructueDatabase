Structure-based Kinases Resources: KPDB, KLPDB, AKPDB, and CIKPDB

The Structure-based Kinase Resources is a comprehensive collection of all published kinase domain structures from UniProt and PDB. It includes:

  • KPDB: Kinase Domain PDB database
  • KLPDB: Kinase Ligand-binding PDB database
  • AKPDB: Apo Kinase PDB database
  • CIKPDB: Covalent Inhibitor-binding Kinase PDB database
